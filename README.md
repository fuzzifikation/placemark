# Placemark

Privacy-first, local-first tool for exploring and organizing photos by where and when they were taken.

Visualize photos on a map using EXIF GPS metadata, filter by date ranges, and organize files — without uploading anything to third-party servers.

**Status:** Early development. Architecture and core concepts are being established.
